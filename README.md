# Platformer
Something fun
## Requirements
* Love2D
## Usage
Change the Path in *run.bat* to your *lovec.exe* then run in the command line\
```./run.bat```
## Controls
* ```a``` or ```left``` = move left
* ```d``` or ```right``` = move right
* ```w``` or ```up``` or ```space``` = jump
* ```e``` = interact or emote
* ```p``` = attack
* ```escape``` = Pause
